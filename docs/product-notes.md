# 产品说明（v5）

## 当前功能

- 首页：人格图鉴 + 维度说明
- 独立测试页：`quiz.html`
- 12 种人格原型
- 12 道抽象搞笑版题目（更强分流）
- 重复维度聚合 + 最近邻匹配

## v5 平衡性修正

### 1) SAFE 改尖
SAFE 现在更像：
- 规范更强
- 风险更保守
- 决策更审慎
- 细节 / 审查意识更明显

### 2) 聚合逻辑改成“多数派”
- L+L → L
- L+M → L
- M+M → M
- H+M → H
- H+H → H
- L+H → M

### 3) 题面更像不同人格在说话
- 甲方扯皮
- AI 替代土木狗
- 锅不在 scope 但闻到糊味
- 入行动机
- 祖传做法 / 老经验 / 现场鬼故事
